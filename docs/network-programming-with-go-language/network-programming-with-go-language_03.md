# 致谢

我想对 Jan Newmarch 为本书的合作所做出的贡献表示感谢。这个项目带给我巨大的成就感，也让我实现了心愿清单上期盼已久的一项。我还要感谢 Eldon Alameda，他总能以深思熟虑的方式让我意识到自己的不足，并给予我中肯的建议。

此外，我要感谢 Apress 的合作伙伴们，包括给予机会的 Steve Anglin 和提供指导帮助我完成此书的 Mark Powers。感谢我在 RX-M 的同事们，包括 Randy Abernethy、Christopher Hanson、Andrew Bassett 和 Anita Wu。我们多年来的工作使我能够参与像本书这样的项目。

最后，我要感谢我的妻子 Julie 和女儿 Charlotte。当我埋头于电脑前时，Julie 管理家庭的能力无人能及，我对此深表感激。Charlotte 的活力、能力和创造力激励着我每天都变得更好。

> —Ronald Petty
