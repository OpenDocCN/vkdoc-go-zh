# 第二版前言

尽管在互联网时代，岁月已悄然流逝，但 Go 语言仍然是程序员的首选语言之一。Go 语言征服了容器技术领域，并持续在云原生开发中备受青睐。Go 力求保持自身特色、向后兼容，同时不断添加像泛型这样的新语言特性。诸如模糊测试等工具的改进，使得开发出更安全的应用程序成为可能。

Go 语言发生了改变，这本书也随之更新。第一版使用的是 Go 1.8，我们现在已更新到 Go 1.18。代码已更新以反映这一新变化。本书中的示例主要旨在展示 Go 网络编程的特定功能，而不会引入不必要的复杂性，例如管理散落在全书中的多个项目或包；相关的代码仓库可在此处找到：[`https://github.com/Apress/network-prog-with-go-2e`](https://github.com/Apress/network-prog-with-go-2e)。

本书的第一版假定读者熟悉 Go 语言，这一版仍保持此设定。我们稍微扩展了本书中愿意讨论的内容，纳入了更多第三方模块、工具和技术。Jan 坚持将重点放在 Go 语言本身，而不被整个生态系统分心，这一点是正确的。

如果你渴望学习如何使用 Go 实现网络编程的概念，我希望这本书能对你有用。
