# 引言

Go 语言已经问世超过 10 年，并且有许多开源项目是用 Go 开发的。本书旨在向你展示如何使用 Go 编写在基于云的系统中有用的各种应用程序。

将应用程序部署到云端是开发人员日常的常规流程。关于云，开发人员会问自己很多问题，例如：

*   容器在云环境中如何工作？
*   云监控应用程序如何知道我的虚拟机还剩多少内存？
*   如何在 Linux 环境中构建高性能网络服务器？
*   如何在部署到云端之前扫描代码，如果包含相关信息则阻止代码部署？

以及许多其他与云相关的问题。

本书讨论了当今云环境中的各类相关主题。方法是对每个主题进行高级概述，然后通过代码细节帮助你理解。本书结合使用了托管在 GitHub 上的开源项目和示例代码。所选的开源项目都与主题相关。你将深入了解该工具及其代码的内部工作方式。
