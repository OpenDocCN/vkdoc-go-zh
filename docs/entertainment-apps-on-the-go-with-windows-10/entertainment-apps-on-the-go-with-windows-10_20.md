# 后记

在本书的引言中，我们提到，在制作每周播出的《数字生活方式秀》的十多年间，我们目睹了许多存储和播放媒体（无论是音乐、视频还是照片）的方式兴起又消亡。我们希望已通过本书证明，Windows 10 有潜力成为处理这些媒体的最佳方式之一。

我们展示了内置应用如何能够管理您的个人媒体收藏，同时还增加了大量功能，例如将您的收藏存储在云端以便随处播放。新的 Windows 商店为购买媒体提供了一站式服务，而 Groove 音乐订阅服务是 Spotify 等服务的强劲竞争对手，尽管遗憾的是没有免费套餐。

我们展示了 Windows 10 如何与苹果、谷歌和亚马逊等其他音乐服务良好协作，甚至还介绍了微软如何在 Android 和 iOS 上提供自己的服务。

我们探讨了 Windows 10 如何在您家中乃至更远的范围内流式传输媒体，甚至可以借助内置的（如果您使用过以前的 Windows 版本，应该很熟悉）Windows Media Player 和第三方服务来实现。

总而言之，我们对 Windows 10 感到非常兴奋；它已经成为最引人注目的媒体解决方案之一，并且随着持续更新的前景以及开发者能够为包括 Xbox 在内的所有 Windows 平台编写应用的能力，它只会变得更好。
