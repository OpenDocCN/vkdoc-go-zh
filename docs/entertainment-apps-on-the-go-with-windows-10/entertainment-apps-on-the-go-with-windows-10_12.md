# 4. 从 Windows 设备流式播放您的媒体收藏

我们在《数字生活方式》节目中经常被问到的一个问题是如何将音乐和视频发送到家中的其他电脑。例如，这可能涉及将音乐从您的主电脑发送到另一房间连接到音响的网络电脑，或者将视频从电脑发送到卧室的平板电脑。这通常被证明颇具挑战性。在本章中，您将了解较旧版本和 Windows 10 如何应对这一挑战。

借助 Windows 10，您可以访问家中其他 Windows 设备（即使它们运行的是较旧版本的 Windows）上存储的音乐和视频。在本章中，您将学习如何设置 Windows 7、Windows 8.1 和 Windows 10，以便向您的 Windows 10 设备提供内容。

有几种不同的选项可用于连接到 Windows 电脑；我们将介绍如何设置文件共享、使用 Windows 家庭组以及使用 Windows Media Player 充当服务器。

**信息**

家庭组和网络共享在 Windows 7（及 Windows Home Server）、Windows 8.1 和 Windows 10 上的工作方式大致相同，因此这些说明适用于您所使用的任何平台。
