# 从 `iTunes` 等应用向 Windows 10 手机传输音乐

你可以通过互联网或直接通过 USB 连接，将音乐从 Windows 10 电脑或平板电脑直接复制到 Windows 手机。

如果使用 USB，你可以通过文件资源管理器复制文件。连接手机后，即可将内容从电脑复制到手机。通过 USB 连接手机，并确保手机已解锁。

然后，你可以使用文件资源管理器浏览文件系统，找到想要复制的音乐。找到某个歌手、专辑或歌曲后，右键单击（或长按）并选择“复制”。

接着，通过 `此电脑` 分区中的文件资源管理器，你会看到你的 Windows 手机；导航到 `音乐` 文件夹，然后选择“粘贴”。这会将选中的内容复制到手机。

你还可以将两个窗口并排放置，通过简单的拖放操作在电脑和手机之间复制文件夹。
