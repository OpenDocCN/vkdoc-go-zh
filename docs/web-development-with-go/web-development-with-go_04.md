# Go 编程语言概述

`Go`，通常被称为`Golang`，是一种通用编程语言，由谷歌于 2009 年 11 月开发。

有多种编程语言可用于编写不同类型的软件系统，其中一些语言已经存在了几十年。一些主流编程语言通过在其新版本中添加新特性而不断演进，每个版本都会发布许多新功能。`C#`和`Java`都在其语言规范中提供了过多的特性。

与此同时，计算机硬件和 IT 基础设施也在不断创新和发展。软件系统使用功能丰富的编程语言编写，但借助它们，我们无法充分利用现代计算机和 IT 基础设施的能力。我们使用的编程语言诞生于单核机器时代，而现在我们使用这些语言为多核机器编写应用程序。

与其他事物一样，计算机编程语言也在不断发展。`Go`是一种进化型语言，它使用简洁实用的编程语言，为现代计算机和 IT 基础设施编写软件系统。在`Go`官网[`https://golang.org/`](https://golang.org/)上，`Go`被定义为：“`Go`是一种开源编程语言，可以轻松构建简单、可靠且高效的软件。”

`Go`旨在解决实际问题，而非学术理论或智力思辨。`Go`是一种实用的编程语言，它忽略了在过去三十年中发展起来的编程语言理论（`PLT`）；它提供了一种简单的编程模型，用于构建高效的软件系统，并内置了对并发的一等支持。`Go`内置的并发特性为你提供了令人兴奋的编程体验，使你能够利用并发编写出高效的软件系统。每种编程语言都有其设计目标。`Go`被设计成一种简单的编程语言，并且它作为一种简单实用的语言表现出色。

`Go`是构建众多创新软件系统的首选语言，包括`Docker`、`Kubernetes`等。许多现有系统（例如 Facebook 的`Parse MBaaS`）正在用`Go`进行重构。我曾协助多家组织成功采用`Go`，由于`Go`的简洁性和实用性，采用过程非常顺利。我相信，当你开发真实的软件系统时，你会对`Go`感到兴奋。

`Go`是一种通用编程语言，可用于构建各种软件系统，包括网络服务器、系统级应用程序、基础设施工具、`DevOps`、原生移动应用、图形处理、物联网（`IoT`）以及机器学习。`Go`还可用于构建原生移动应用程序，我预测在不久的将来，`Go`将成为构建原生安卓应用的绝佳选择。

`Go`是构建 Web 应用程序和后端`API`的绝佳语言选择。我强烈推荐使用`Go`构建可大规模扩展的后端`RESTful API`。我预测，在企业中，`Go`将成为构建后端`RESTful API`的首选语言，而`RESTful API`是构建现代业务应用程序的骨干，尤其是在这个移动时代。

在本书中，我假设你至少掌握一种编程语言，并具备一定的 Web 编程经验。如果你对`Go`有初步了解，将有助于你阅读本书。如果你完全是`Go`新手，我建议你在开始阅读本书之前先学习以下教程：[`http://tour.golang.org/welcome/1`](http://tour.golang.org/welcome/1)。

在学习语言基础时，我建议你查阅`Go`文档的以下部分：[`https://golang.org/doc/effective_go.html`](https://golang.org/doc/effective_go.html)。

本书的主要重点是使用`Go`编程语言进行 Web 开发。在深入探讨 Web 开发之前，本书会快速介绍语言基础和并发特性，但不会深入探讨，尤其是在并发方面。如果你想在实际应用中有效利用并发，你应该花些时间深入探索。我推荐以下资源来学习更多关于并发与并行的知识：[`http://blog.golang.org/concurrency-is-not-parallelism`](http://blog.golang.org/concurrency-is-not-parallelism)。

本书探讨了`Go` Web 编程的各个方面，并重点提供了实用代码。第 9 章“构建 RESTful 服务”可以帮助你开始使用`Go`开发真实的`API`。
