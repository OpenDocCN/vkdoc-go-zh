# 1. 简介

Go（又称 GoLang）是由谷歌于 2007 年开发的一门编程语言，其设计以简洁和高效为核心。Go 的设计目标是兼具像 Python 这类动态类型高级语言的简洁可读语法，同时拥有像 C/C++ 这类静态类型低级语言的稳定性和效率。在谷歌的支持下，Go 作为一门开源编程语言，更易于学习和上手。它内置了对并发的支持，拥有强大的标准库、类型安全、多种内置类型、动态类型能力、垃圾回收机制，以及诸如键值映射和可变长数组等其他高级特性。Go 能让你充分利用多核处理器的强大能力，从而开发出运行更快的程序。

## GoLang 是静态类型还是编译型语言？

任何程序员对一门编程语言的自然关注点在于，该语言是编译型还是解释型。JavaScript 就是解释型语言的一个例子。网页浏览器可以直接读取 JavaScript 源代码并在运行时执行，无需任何预计算。即使某些解释环境可以生成诸如字节码之类的中间格式，也不涉及预计算步骤。

### 编译型编程语言

与解释型语言相反，编译型语言的源代码会被转换成特定于操作系统的格式。与 C 和 C++ 一样，Go 是一门编译型语言。然而，与将源代码编译成能够在不同操作系统上运行格式的 Java 不同，Go 的源代码会被编译成特定于操作系统的格式。这意味着代码仅能在其编译时针对的操作系统上执行。

### 静态类型语言

Go 是一门静态类型语言。这意味着它的变量必须具有特定的类型。不过，并不总是必须显式声明变量。编译器可以推断类型；然而，在编译时，类型总是已知的。

无需重新编译即可运行源代码文件的能力有时会让程序员认为 Go 是一门解释型语言。真正的奥秘在于后台，源代码被编译成一种临时的可执行形式。需要用到“Go”这个编译工具来构建分发给用户的应用程序。这是因为编译后的可执行文件是特定于操作系统的格式。每个 Go 应用程序都包含一个静态链接的运行时。在应用程序的编译过程中，运行时组件会被打包进可执行格式中。这就是为什么使用 Go 语言构建的应用程序大小相比其源代码要大一些。由于不涉及外部虚拟机，因此有必要包含一个运行时包。

## Go 是一门面向对象的编程语言吗？

Go 并非一门面向对象的编程语言。尽管如此，你会在 Go 语言中发现某些重要的面向对象特性。

*   可以定义自定义接口。在 Go 中，接口可以被视为一种契约，它规定了一组必须由任何实现该接口的程序实现的功能。
*   可以定义自定义类型。自定义类型也可以拥有自己的方法（函数）。值得注意的是，在 Go 中，几乎所有东西都是类型，并且每个类型至少实现一个接口。
*   可以定义自定义结构体（数据结构），包含成员字段和方法。

## 使 GoLang 成为编程首选的功能特性

Go 的其他使其在领先公司中广受欢迎的功能包括：

*   Go 支持与动态语言提供的相同的环境自适应模式。例如，它支持类型推断。这意味着 `intVariable:= 123` 是声明一个名为 `intVariable`、值为 `123` 的 `int` 类型变量的有效方式。
*   Go 的编译速度快得多。
*   Go 支持并发。例如，它提供了 `select` 语句、*goroutines*（类似于轻量级进程）以及通道。
*   Go 程序更加简单、简洁，并相对更安全。
*   Go 支持类型嵌入和接口。
*   无需外部依赖，Go 支持生成静态链接的原生二进制文件。

## GoLang 有意排除的功能特性

为了保持 Go 的简单和简洁，一些类似编程语言通常支持的功能特性被 Go 语言排除在外。省略的主要原因在于，Go 的创造者认为所有这些因素会使编程语言变得越来越难以理解和容易出错。一些被有意排除的功能如下：

*   类型继承（没有类）
*   泛型编程
*   方法或运算符重载
*   指针运算
*   结构化异常处理
*   隐式数值转换
*   断言
*   包之间的循环依赖

## Go 程序

一个 Go 程序可以短至三行，也可长达数百万行。Go 程序被写入扩展名为 `.go` 的文本文件中，例如 `myProgram.go`。你可以使用任何你喜欢的文本编辑器（如 vim、vi、sublime、vs code 等）来编写 Go 程序。为了高效管理依赖，Go 程序使用包来构建。Go 编程实现采用传统的编译和链接模型来生成可执行的二进制文件。



## 摘要

Go 编程语言由谷歌于 2007 年开发。它被设计为一种以简洁和快速为焦点的编译型语言。Go 的设计目标是创建一种具有动态类型高级编程语言简洁可读语法，同时兼具静态类型低级语言的稳定性和效率的编程语言。它是一种开源编程语言，相对更容易学习和上手。Go 内置了对并发、强大的标准库、类型安全、多种内置类型、动态类型能力、垃圾回收以及多项其他高级特性的支持，包括键值映射和可变长数组。

此外，Go 还能让你利用多核处理器提供的能力，从而实现运行更快的程序。关于 Go，需要记住的一个重要方面是它不是一种面向对象的编程语言。不过，你在 Go 语言中确实能找到某些重要的面向对象特性，比如可以自定义接口、类型和结构体（数据结构）。

为了达到其设计目标，Go 语言排除了一些特性。排除这些特性的主要原因在于，Go 的创造者认为这些因素都会让编程语言越来越难以理解且容易出错。
