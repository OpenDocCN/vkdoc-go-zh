## 下载

### Docker

```
docker pull apachecn0/apachecn-golang-zh-pt2
docker run -tid -p <port>:80 apachecn0/apachecn-golang-zh-pt2
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install apachecn-golang-zh-pt2
apachecn-golang-zh-pt2 <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-golang-zh-pt2
apachecn-golang-zh-pt2 <port>
# 访问 http://localhost:{port} 查看文档
```
